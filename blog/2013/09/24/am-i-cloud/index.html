Am I "Cloud"

Sep 24th, 2013

I read an article, that is about the demanding for Cloud Engineer has significantly increased over the past few years.

The first thing that came to my mind, what is Cloud Engineer?

After google around, it seems the first thing that you need to know what is SaaS, Software as a Service.

With the emerging and maturing of CSS3, HTML5 and JavaScript, there has been a lot of SaaS providers running now in the market, popular providers such as GitHub, Google Doc, Google Drive, DropBox and etc.

Now, I am asking myself, am I Cloud?

For one second, I wish to be the Cloud Strife in Final Fantasy 7, cool, smart, strong and brave.

I have been using DropBox and Box as my storage media.

I have using GitHub as my blog, can I call myself Cloud Blogger?

I am using Google Doc for my office work.

Well, I guess I am only considered as Cloud …. user and not a Cloud Engineer, hey, wait a second…., I am developing my game on AWS.

I would definitely dig more into Cloud Engineer as I want to be Cloud.
